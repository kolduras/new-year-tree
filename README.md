## New Year tree - JavaScript demo
![Preview](tree.png)

Happy New Year 2026!

The demo can be seen on the [html-preview.github.io](https://html-preview.github.io/?url=https://github.com/kolduras/new-year-tree/blob/main/new-year-tree-build/new-year-tree.html) website or built from source code.

Build dependencies:
* bash
* Node.js

### Quick start
```
$ git clone git@github.com:kolduras/new-year-tree.git
$ cd new-year-tree
$ npm install && npm start
```
